# Lampa TV Plugin — Custom Bookmarks Categories

Локальний плагін для Lampa TV, який додає користувацькі категорії закладок:
- Створення, перейменування, видалення категорій
- Додавання/видалення елементів (movie/tv)
- Перегляд елементів у категоріях
- Локальне збереження в Storage (без Trakt)

## Встановлення

1. Відкрийте Lampa TV → Налаштування → Плагіни.
2. Додайте URL сирого JS-файлу:
3. Плагін з’явиться у списку плагінів. Натисніть, щоб відкрити "Мої категорії".

## Використання

- У "Мої категорії" натисніть "Створити категорію".
- Відкрийте категорію, додавайте елементи вручну або через кнопки/контекст (якщо підтримує ваша збірка Lampa).
- На екрані повної інформації про фільм/серіал має з’явитися кнопка "В закладку (категорія)". Якщо її немає — це залежить від API вашої збірки; використовуйте ручне додавання.

## Обмеження

- Все зберігається локально: інші пристрої не синхронізуються.
- Деякі “гачки” (кнопка на екрані деталей, пункт контекстного меню картки) залежать від реалізації Lampa у вашій збірці. Якщо вони недоступні, використовуйте екран "Мої категорії".

## Безпека

- Плагін — чистий JS без сторонніх залежностей.
- Перевіряйте URL — використовуйте лише свій GitHub.
